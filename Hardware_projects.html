1. Infrared Security Alarm

This project involves building a security alarm circuit using basic circuits utilizing diodes and opamps. The circuit uses an infrared LED as a light source and a phototransistor to detect the light emitted by the LED. The opamp circuits used do not use negative feedback and instead use positive feedback or no feedback at all. The circuit includes a comparator, Schmitt Trigger, and astable multivibrator. 2. DC Motor Driver Circuit

Learnt about the large-signal operation of bipolar junction transistors (BJTs) by designing and building a DC motor driver using BJTs as switches. Covered the theory of large- and small-signal operation of transistors. Learnt about DC motors and how they work. Built a DC motor driver circuit using BJTs and varied the motor speed by varying the duty cycle of a PWM signal. Controlled the motor's direction using an H-bridge.
