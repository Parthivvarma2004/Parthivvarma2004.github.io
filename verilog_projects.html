1. Enhanced Processor

Created an enhanced processor capable of performing read/write operations using memory or other devices. Machine code could be entered into the processor and then executed. The processor was capable of performing addition, subtraction and branching. Furthermore, it could perform 'mv' (Move), 'and' , 'ld' (Load) and 'st' (Store) operations.
It was implemented using a finite state machine that connected an Arithmetic Logic Unit, Bus wires and an Instruction Register.

2. Bouncing Box Animation

Designed and implemented a circuit utilizing a DE1-SoC FPGA board with a VGA adapter module to produce and display a bouncing box animation on screen.
It was implemented using a finite state machine.
Furthermore, additional functionality was added to code to change box color.
